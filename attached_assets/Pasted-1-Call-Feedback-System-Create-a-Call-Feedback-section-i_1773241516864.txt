1. Call Feedback System

Create a Call Feedback section inside the portal.

Admin should be able to:

Upload call feedback notes after every call

Upload documents or files related to the call

Save written feedback from the session

Upload feedback even after the call has ended

Clients should be able to:

View call feedback uploaded by the admin

View documents uploaded for that specific call

Submit their own feedback about the call

Write what they learned from the session

This should create a two-way feedback system between admin and client.

2. Content Tracking System

Inside this section, create three subsections:

Content Tracking

Sales Tracking (Coming Soon)

Ad Metrics (Coming Soon)

Examples:

March 2026
April 2026
May 2026

Example entry:

12 March → Reel posted
15 March → Carousel posted
18 March → YouTube video posted

When a client logs content, the system should automatically remind them 48–96 hours later to update metrics.

Example:

If a reel was logged on 12 March, the system should remind the client to update:

Views

Engagement

Followers gained
