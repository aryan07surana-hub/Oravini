AI Report Generator (Date Range Selection)

Inside the Tracking section, there is currently a button called "Generate AI Report for March".

Remove this static wording.

Instead create a "Generate AI Report" button.

When the user clicks this button, open a date selection interface with a calendar.

The user should be able to:

Select a custom date range

Choose start date and end date

Select common quick options:

1 Week

2 Weeks

4 Weeks

The calendar should display the full month (for example March 1–31).

Users can choose any date range themselves.

Example:

March 3 → March 10

March 1 → March 14

March 7 → March 28

The AI report must analyze only the log posts within the selected date range.

Important rule:
The Generate AI Report button should only be enabled if the user has logged posts inside the tracking system.

If there are no logged posts, disable the button or show a message:
"Log posts first to generate an AI report."

AI Report Data Source

When generating the report, the AI must analyze the logged content data, including:

Views

Likes

Comments

Saves

Engagement rate

Posting frequency

The report should generate insights such as:

Best performing content

Engagement trends

Growth opportunities

Recommendations

The report should be based only on the selected date range.

Reel Tracking Controls (Edit / Delete / Regenerate)

For every reel that is logged in the Content Metrics tracking system, display the following buttons:

Edit

Delete

Regenerate

Instagram Scraper API Integration

Connect the Instagram Scraper API to the Sync Data button.

When the user clicks Sync Data for a specific reel:

The system should fetch updated data for that reel including:

Views

Likes

Comments

Saves

The system should update the metrics automatically.

This should work separately for:

Initial metrics

After 2 weeks

After 4 weeks

Sync Status System

Each reel should show a Sync Status indicator.

Example:

Last Synced:

Initial → Synced March 3

2 Weeks → Synced March 17

4 Weeks → Not Synced

When the user presses Sync, the system should:

Fetch the latest data from the Instagram scraper

Update the metrics

Record the sync time

Update the engagement calculations

Engagement Rate Update

After syncing the data, automatically recalculate engagement rate using:

(likes + comments + saves) / views * 100

This should update the reel’s metrics dashboard instantly.

Reel-Specific Sync

The Sync button must work per individual reel.

Example:

Reel #1 → Sync only that reel
Reel #2 → Sync only that reel

Do not sync all reels at once.

Each reel must be handled independently.

UI / UX Requirements

The regenerate and sync system should:

Be clean and simple

Use status indicators

Show loading animation while syncing

Update metrics instantly after syncing

Use a premium dashboard design with clear buttons and status indicators.

This system should feel smooth and easy for users managing their content tracking.
