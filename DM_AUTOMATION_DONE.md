## 🎯 HOW TO USE

### **1. Scheduled Broadcasts**
- Go to **Broadcasts** tab
- Click "Schedule Broadcast"
- Set date/time
- Target audience (optional)
- Write message
- Schedule!

### **2. Contact Tagging**
- Open any lead
- Scroll to "Tags" section
- Type tag name
- Click "Add"
- Filter leads by tags in main view

### **3. Webhooks (Zapier/Make)**
- Go to **Webhooks** tab
- Click "Add Webhook"
- Enter webhook URL
- Select trigger event
- Test webhook
- Activate!

### **4. Click Tracking**
- Go to **Tracking** tab
- Click "Create Link"
- Enter destination URL
- Copy tracking link
- Share in DMs
- View click stats

### **5. Custom Fields**
- Go to **Custom Fields** tab
- Click "Add Field"
- Enter field name
- Select type
- Use in lead details

### **6. AI Lead Scoring**
- Open any lead
- Scroll to "AI Lead Score"
- Click "Score Lead"
- View score + reasoning
- Re-score anytime

### **7. Welcome DM**
- Go to **Settings** tab
- Enable "Welcome DM"
- Set delay (optional)
- Write message
- Use variables: {{first_name}}, {{name}}, {{instagram}}
- Save!

### **8. Opt-Out Management**
- Open any lead
- Scroll to "Subscription Status"
- Toggle switch
- Opted-out leads excluded from broadcasts

### **9. CSV Export**
- Go to **Leads** tab
- Click "Export CSV"
- Download file
- Open in Excel/Sheets
