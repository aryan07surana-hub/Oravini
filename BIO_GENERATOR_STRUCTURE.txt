=============================================================================
                            TOOL DETAILS
=============================================================================

TOOL 1: GENERATE FROM SCRATCH
Route: /tools/bio-generator/generate
Component: BioGenerateFromScratch.tsx
API: POST /api/tools/bio-generator/generate

INPUT:
- Niche (pre-filled from onboarding)
- Platform (Instagram/Twitter/LinkedIn)
- What you do
- Who you help
- Social proof
- CTA
- Link

OUTPUT:
- 3 variations for Line 1 (Attract/Repel)
- 3 variations for Line 2 (Social Proof)
- 3 variations for Line 3 (CTA)
- Line 4 (Link - user provided)
- Profile picture tips
- Mix-and-match interface
- Character count validation

-----------------------------------------------------------------------------

TOOL 2: IMPROVE EXISTING BIO
Route: /tools/bio-generator/improve
Component: BioImprove.tsx
API: POST /api/tools/bio-generator/improve

INPUT:
- Current bio (paste)
- Platform selection

OUTPUT:
- Overall score (0-100)
- Score breakdown:
  * Clarity (20 pts)
  * Hook Strength (20 pts)
  * CTA Effectiveness (20 pts)
  * Character Efficiency (20 pts)
  * Niche Specificity (20 pts)
- 3 improved versions
- What changed list
- Character limit validation

-----------------------------------------------------------------------------

- Character count validation

-----------------------------------------------------------------------------

TOOL 4: COMPETITOR ANALYSIS
Route: /tools/bio-generator/competitor
Component: BioCompetitor.tsx
API: POST /api/tools/bio-generator/analyze-competitors

INPUT:
- 2-5 competitor bios (paste)
- Platform selection
- User niche (optional)

OUTPUT:
- Pattern analysis:
  * Common structure
  * Tone analysis
  * Keyword frequency
  * CTA patterns
  * Average length
- Key insights
- 3 unique generated bios
- Character limit validation
